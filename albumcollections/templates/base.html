ATTENTION: I will be taking this site down on May 1st, 2024

I have decided to stop maintaining this application. This personal project has been a way for me to explore how to organize my albums while continuing to utilize the convenience of Spotify, and exposing what I've created to others interested in using an application like this. But I've now discovered more cost-effective, performant and flexible methods.

Here are a few suggestions for the next step in your spotify album collection journey:

  • Check out https://albumsortify.fr, another web app someone has created for organizing spotify albums
  • If you'd like to continue following my methods, I'm now using obsidian to store and organize my albums as basic text markdown files. I created a custom plugin to be able to easily import spotify album data into my notes.
  • If you really want to continue using this application, you can do that by pulling down the code and running it yourself.

    • If you'd like to chat, feel free to reach out to me at nickknaian@gmail.com
